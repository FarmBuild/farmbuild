FarmBuild - Agriculture science web service API distribution model

DEDJTR is seeking to provide software and cloud based agriculture service companies access to research based outputs and key agricultural datasets as on-line functions that can be built into service offerings for their client base across key Victorian agriculture industries of horticulture, dairy, red meat and grains. The ability for companies to access and integrate science based algorithms and information with their systems means value can be added to the ARD knowledge base and provides ARD with access to the rapidly expanding cloud based farm management services environment.
